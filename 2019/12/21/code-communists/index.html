Code Communists

I’ve had an interest in Communist ideology for some time. After I uncovered that nodejs had a Communist problem I decided to see where else the evidence pointed. Did Communist ideology really permeate the industry as some people claimed?

This month I created a new larger investigation called Code Communists which goes over some of the ideology generally and then organizations like Microsoft, Google, GitHub, and Netlify more specifically. It has been an illuminating experience for me and I hope you get something out of it too.
